🎤 WeatherBot Voice Assistant

Test the voice assistant by clicking the microphone button or using the examples below.

Try These Voice Commands:

"What's the weather in London?"
Get current weather for a specific city
"How's the weather in Tokyo tomorrow?"
Ask about weather for a specific time
"What's the temperature in New York?"
Ask specifically about temperature
"Weather forecast for Paris tonight"
Get forecast for evening/night
"Help"
Get assistance with available commands
"Goodbye"
End the voice assistant session

Ready to listen

How It Works:

  • Speech Recognition: Uses the Web Speech API to convert your voice to text
  • Natural Language Processing: Understands weather-related queries and extracts city names and time references
  • Weather API Integration: Fetches real weather data from OpenWeatherMap
  • Text-to-Speech: Converts weather responses back to speech

Browser Compatibility:

Supported: Chrome, Edge, Safari (with limitations)

Not Supported: Firefox (requires HTTPS for speech recognition)

Note: Voice assistant works best with HTTPS or localhost
