How to Add Amazon Affiliate Products

  1. Go to the Amazon product page you want to add
  2. Copy the ASIN from the URL (e.g., B08N5WRWNW)
  3. Get the product image URL by right-clicking the main product image
  4. Fill out the form below with product details
  5. Make sure your affiliate tag is set in affiliate_products.py
  6. The affiliate URL will be generated automatically using your ASIN
